proc
remove_var_copy
(target_copy)

Removes a var copy from the global list that contains them, effectively the same as deletion. Automatically executes if a text argument is given, assuming it is the name of the copy to delete.

Args: target_copy: Text. The proc will assume this is the key, or the name, of the var copy, and will search GLOB.var_copies for it's value.

Spawns a atom on the location of the user, using the type variable within GLOB.var_copies[target_copy], using the target_copy argument. Then searches said list for any variables. If any variables are present, they will be applied to the newly created atom.
