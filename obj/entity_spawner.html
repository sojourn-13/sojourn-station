spawn_messageUsed as \the [src] [spawn_message] [spawned_mob]
spawn_offsetAssociative. Any value here will shift the spawns that many tiles away, using coordinates. X = horizontal shifting, Y = vertical shifting, Z = z-level shifting. Format: list(x = offset, y = offset, z = offset). If a value is null, it defaults to 0, AKA no offset. Set the entire var to null to disable.
spawn_overrideAssociative. If this is not null, anything spawned will be spawned at these coordinates. X = horizontal, Y = vertical, Z = z-level. Format: list(x = coordinate, y = coordinate, z = coordinate). If a value is null, it defaults to src's respective coordinate. Set the entire var to null to disable.
spawn_per_spawnHow many mobs do we spawn per successful spawn?
spawn_probabilityused in prob() to determine if this will spawn a mob this process tick
