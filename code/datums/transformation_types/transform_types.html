code/datums/transformation_types/transform_types.dm

/datum/transform_typeDatum-based transforms, used for compatability between transform types. Ideally will replace all matrix/transform replacements and transforms we have. To use this, you have two options: One, add a new transform_type to types.dm, and apply it with add_transformation_type or add_initial_transforms. This allows full functionality. The second is to use a instance of /transform_type/modular and pass the variables in add_transformation_type through args. This isn't reccomended, but if you dont want to make A new datum, this works decently.
